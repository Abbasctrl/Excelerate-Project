# Skills Development App - User Guide

## Welcome!

This Flutter application helps you discover and develop skills in various domains including UI/UX Design, Software Development, and Graphic Design.

## How to Use the App

### 1. Login Screen
- **Location**: First screen when you open the app
- **How to login**: 
  - Enter any email address (must contain @)
  - Enter a password (at least 6 characters)
  - Click "Login" button
- **Example credentials**:
  - Email: `learner@example.com`
  - Password: `password123`

### 2. Home Dashboard
After logging in, you'll see:
- **Welcome message** with your username
- **Category cards** showing:
  - UI/UX Design (3 programs)
  - Development (3 programs)
  - Design (2 programs)
- **Popular programs** list with top courses
- **Navigation options**:
  - Feedback icon (top right) - Submit feedback
  - Logout icon (top right) - Sign out
  - "View All" - See all programs

### 3. Program Listing Screen
- **Filter by category**: Tap category chips at the top
  - All, UI/UX Design, Development, Design
- **View program details**: Tap any program card
- **Information shown**:
  - Program title and description
  - Difficulty level (Beginner/Intermediate/Advanced)
  - Rating, instructor, duration, enrolled students

### 4. Program Details Screen
When you tap a program, you'll see:
- **Program overview**: Full description
- **Duration and instructor** information
- **Learning modules**: Step-by-step curriculum
- **Skills you'll gain**: List of competencies
- **Enroll button**: 
  - Tap "Enroll Now" to join the program
  - Tap "Unenroll" to leave the program

### 5. Feedback Form
Access from the home screen (feedback icon):
- **Required fields**:
  - Your name
  - Email address
  - Subject
  - Message (at least 10 characters)
  - Rating (1-5 stars)
- **Submit**: Tap "Submit Feedback" when complete
- **Confirmation**: You'll see a success message
