# Skills Development App - Replit Project

## Overview
A complete Flutter/Dart skills development application with login, program browsing, detailed program views, and feedback functionality. Built for helping users discover and develop skills in UI/UX Design, Development, and Graphic Design.

## Project Status
Created: October 30, 2025
Status: Complete - All MVP features implemented

## Architecture
- **Framework**: Flutter 3.24 with Dart
- **State Management**: Provider pattern
- **Data Storage**: SharedPreferences for user sessions
- **Data Source**: Mock JSON API with local asset files
- **Navigation**: Material PageRoute

## Key Screens
1. Login Screen - Form validation, mock authentication
2. Home Screen - Categories, popular programs
3. Program Listing - Filter by category
4. Program Details - Enrollment, modules, skills
5. Feedback Form - Rating system, validation

## Data Structure
- 8 skill development programs in JSON
- 3 categories: UI/UX Design, Development, Design
- Mock authentication service
- Feedback submission system
